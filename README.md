# IDF_Operation

## תיאור
מערכת סימולציה המדמה אינטראקציה בין צה"ל לארגוני טרור (כגון חמאס) באמצעות קונסולת C#.

## מאפיינים עיקריים
- ניהול יחידות צה”ל, כלי נשק, ושדה קרב.
- הצגת תפריטים והדמיית פעולות התקפה/מודיעין.
- הרחבה קלה להוספת כוחות, כלי נשק ותסריטים נוספים.

## דרישות מערכת
- .NET Framework 4.7.2 ומעלה או .NET Core (תלוי בקובץ הפרויקט)
- Visual Studio 2019 ומעלה / dotnet CLI

## התקנה והרצה
1. שיבוט הרפוזיטורי:
   ```sh
   git clone https://github.com/yakir-uzan/IDF_Operation.git
   cd IDF_Operation
   ```
2. בניה והרצה דרך Visual Studio או:
   ```sh
   dotnet build
   dotnet run
   ```

## דוגמת שימוש
בעת הריצה מוצג תפריט לבחירת פעולה (למשל: ביצוע תקיפה, צפייה במידע מודיעיני). יש לעקוב אחרי ההנחיות במסך הקונסולה.

## בדיקות
בהמשך יתווספו בדיקות יוניט בסיסיות (תיקיית `Tests`). כרגע אין.
